 SYC_AssetSystem
  
  当前是一个 桌面端资产管理系统雏形,具备接入 Everything 搜索引擎 的能力。
使用技术栈：
  
  .NET 8

Avalonia 11.3.0

CommunityToolkit.Mvvm

EverythingNetCore 2.0.0



1. **应用名称**
SYC Asset System（资产服务管理系统）

2. **应用简介**
SYC Asset System 是一款基于 .NET 8 与 Avalonia 框架开发的桌面端资产服务管理工具，主要用于对项目资产目录进行统一浏览、树形展示和快速检索。系统面向本地磁盘或共享网络盘中的资产文件夹，帮助用户提升资产定位效率，减少手工查找时间。

3. **应用目标**

本系统旨在解决项目资产目录层级复杂、文件数量多、人工查找效率低的问题，为用户提供一个轻量、直观、可扩展的资产查询与浏览入口。

4. **适用场景**
项目资源目录浏览
共享盘资产快速定位
文件夹结构可视化管理
按名称检索模型、贴图、文档及其他资源文件
资产目录日常维护与辅助查询
